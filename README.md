## How to install

---

1. Clone the repo using this command on the terminal:

```
git clone https://github.com/PedroXA/my-react-portfolio.git
```

## Executing the project

---
2. To execute the project follow the steps:

```
cd <folder-name>
```

3. Run the application:

```
npm run dev
```
